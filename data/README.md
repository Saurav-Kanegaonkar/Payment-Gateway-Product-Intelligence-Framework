# Data Sources

This folder contains synthetic data for the payment gateway product intelligence artifact. The data is generated by `scripts/generate_gateway_artifact.py` with a fixed seed so the analysis can be reproduced.

The synthetic structure is modeled on embedded payment gateway operations: partner portfolios, payment channels, payment methods, processor routing, transaction lifecycle metrics, settlement delay, ACH return behavior, reconciliation variance, webhook quality, incident routing, and experiment planning.

## Tables

- `partners.csv`: Partner portfolio metadata and operating ownership.
- `transaction_daily.csv`: Daily partner x channel x payment method x processor transaction metrics.
- `data_quality_incidents.csv`: Incident records generated from data quality, settlement, ACH, webhook, duplicate event, and processor latency signals.
- `reconciliation_checks.csv`: Weekly gateway and ledger comparison records for card and ACH settlement workflows.
- `experiments.csv`: Product experiment candidates with expected lift, readiness, confidence, and OKR linkage.

The data does not represent real company or merchant performance.
